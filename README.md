# Deepfakeimage
This project uses Convolutional Neural Networks (CNNs) built with TensorFlow and Keras to detect AI-generated (fake) images and distinguish them from real ones. It was trained on 80,000+ facial images, with preprocessing and augmentation for improved accuracy.
